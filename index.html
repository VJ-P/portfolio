easyDocs

easyDocs

Technology Used: Python, Django, SQLite, HTML, CSS, JavaScript

This project was in collaboration with a team of 4 at DeltaHacks 5. Doctors listed the burden of paperwork as their top concern in the Medical Economics 89th Annual Physician Report (Dahl, 2018). Our goal was to create an application that would help reduce the amount of inefficient paper used in the healthcare industry.

We did this by creating a web application using the Python Django framework. The web application would assist doctors by keeping them updated on their patients and assist in taking notes. The core of our application was a complex SQL database that we designed. The database kept track of several components such as appointments, patient personal information, healthcare providers, conditions/diseases, current prescriptions, family history, etc.

The database was then connected to a simple interface that healthcare providers could access which would allow them to edit patient information, manage appointments, and export the patient information into an easy-to-read Microsoft Word document. Not only did the Word document provide patient information, but also warned the doctor to check for certain diseases based on medical/family history and warned of medication conflicts.

NBA Award Analysis

NBA Award Analysis

Technology Used: Python, Tableau, MySQL

As someone who is a huge basketball fan, I was excited to see the way advanced analytics/stats was having an impact on the game and the way it was being played. I wanted to use the analytics to do some of my own research and determine who the players are that are moving under the radar as well as players who were rated higher than their level of play indicated.

For this project I scraped a basketball statistics website using the BeautifulSoup Python Package. I then filtered out players from the data who had small sample sizes of playing time (injuries/non-rotation players) and stored the data into a MySQL database. From there the data was visualized using Tableau to identify trends and predict the end of season award winners.

Focus

Focus

Technology Used: Python, Windows

This is a simple Python script that I developed to run in the background of Windows as a process during a set schedule. The script blocks access to certain distracting websites such as Facebook, Twitter, Reddit, Youtube, etc. so that I can fully focus on my work (school or personal). Not only was this useful for focusing on and completing work, the program helped reduce my stress as well I was spending less time on these social medias, and more time being productive.

This program works by manipulating the 'hosts' file in Windows and redirecting each website to an error page IP when someone tries to access it in the set time range.

Define

Define

Technology Used: Python

This project is a command-line application that I developed to quickly provide the definition of words when needed. The program utilizes a JSON data set which contains all the words as the 'key', and the definitions as the 'value'. The program then requires an input from the user for the word that needs to be defined. I also utilized the difflib library in Python to find the closest match in the case that the user inputted a typo for the word they were trying to define.

Autobot

Autobot

Technology Used: Arduino

This project was in collaboration with 2 other engineers at the McMaster Sumobot Competition. The bot was constructed using using electrical components (DC motor, H bridge, protoboard, ultrasonic sensor, etc.) and controlled using an Arduino Uno board.

For this project I developed an Arduino program to detect objects in its surroundings with an ultrasonic sensor, and adapt movement based on the environment. The program had repeatable pattern when it was trying to sense/detect the other robot but, move into attack position when it detected something within a certain range. The bot would also override whatever function it was running if the colour sensors detected 'white', meaning that it was leaving the ring.

My biggest takeaway from this project was how similar concepts and technology could applied to bigger projects such as self driving cars.
